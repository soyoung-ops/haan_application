# 📚 SAT 빈출 단어 학습 앱

Streamlit을 사용하여 만든 SAT 빈출 영어 단어 학습 웹 애플리케이션입니다.

## 🚀 시작하기

### 1. 필요한 패키지 설치

```bash
pip install -r requirements.txt
```

### 2. 앱 실행

```bash
streamlit run app.py
```

브라우저에서 자동으로 앱이 열립니다. (일반적으로 `http://localhost:8501`)

## ✨ 주요 기능

### 📖 학습 모드
- **단어 카드 형식 학습**: 단어, 의미, 예문을 카드 형식으로 학습
- **난이도 필터**: High/Medium 난이도별 필터링
- **학습 진도 추적**: 학습한 단어를 표시하고 진도를 추적
- **단어 검색**: 단어나 의미로 검색 가능
- **랜덤 학습**: 랜덤으로 단어 선택

### 🧩 퀴즈 모드
- **다양한 문제 수**: 5~20문제 중 선택 가능
- **객관식 퀴즈**: 4지선다 형식
- **즉시 피드백**: 답 확인 시 정답/오답 표시
- **결과 분석**: 점수, 등급, 상세 결과 제공

### 📊 통계 기능
- 학습한 단어 수 추적
- 진도율 시각화
- 퀴즈 점수 및 등급 표시

```

## 🎨 특징

- **아름다운 UI**: 그라데이션 카드와 현대적인 디자인
- **반응형 레이아웃**: 다양한 화면 크기에 대응
- **사용자 친화적**: 직관적인 인터페이스
- **40개 이상의 SAT 빈출 단어**: 실제 SAT 시험에 자주 나오는 단어 포함

## 📝 사용 방법

1. **학습 모드**:
   - 사이드바에서 난이도를 선택하고 "필터 적용" 클릭
   - 단어 카드를 보며 학습
   - "✅ 학습 완료" 버튼으로 진도 업데이트
   - 이전/다음 버튼으로 단어 이동

2. **퀴즈 모드**:
   - 사이드바에서 "🧩 퀴즈 모드" 선택
   - 문제 수를 선택하고 "🔄 새 퀴즈 시작" 클릭
   - 각 문제에 답을 선택하고 "✅ 답 확인" 클릭
   - 모든 문제를 완료하면 결과 확인

## 🔧 커스터마이징

`words_data.py` 파일을 수정하여 단어를 추가하거나 수정할 수 있습니다.

    "word": "단어",

```
